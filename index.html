HTML

  • Refers to links that connect web pages to each other, either within a single site or between sites.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML uses "markup" to annotate text, images and other content for display in a web browser.

CSS

  • CSS is called the Cascading Style Sheet language and is used to style elements written in a markup language such as HTML.
  • With CSS you no longer have to repeatedly write down how individual elements look.
  • CSS is a very powerful tool that makes it possible to create various functionalities.

  • git checkout -b branch-name: creates a new branch and switches to it
